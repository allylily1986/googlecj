# Quick Insight Notes (Chrome 扩展)

Quick Insight Notes 是一款针对 Chrome 浏览器的快捷笔记扩展，帮助你在任意网页中记录学习心得与临时灵感。按下默认快捷键 `Alt+N` 就能唤醒浮动面板，支持 Markdown 文本、截图附件以及分类管理。

## 安装与运行

1. 打开 Chrome 扩展页面（`chrome://extensions`），开启 **开发者模式**。
2. 选择 **加载已解压的扩展程序**，指向本仓库的 `extension/` 目录。
3. 加载后可在地址栏右侧找到扩展图标，点击打开主界面。

> 提示：快捷键可以在扩展详情页 > 键盘快捷键中重新设置。

## 核心功能

- **浮动笔记面板**：任意网页按 `Alt+N` 呼出，可拖拽定位，支持浅/深色主题。
- **分类管理**：在主界面筛选、在设置页新增/重命名/删除分类，并指定默认分类。
- **截图与附件**：一键截取当前可视区域，或上传本地图片，随笔记一同保存。
- **导出能力**：单条笔记可导出 Markdown 或内嵌截图的 DOCX；批量导出会打包成 ZIP。
- **备份恢复**：设置页支持导出/导入 JSON 备份（含笔记、分类及设置）。

## 项目结构

```
extension/
  manifest.json        # Chrome 扩展配置
  service_worker.js    # 后台服务（命令、存储、导出等）
  common/              # 共享工具、存储与导出逻辑
  content/panel.*      # 网页内浮动面板逻辑与样式
  ui/list.*            # 工具栏弹出主界面（列表视图）
  ui/options.*         # 设置页面
```

## 开发提示

- 修改后可在扩展页面点击 **重新加载**，刷新当前页面或弹窗即可测试。
- 截图功能依赖 `tabs` 与 `activeTab` 权限，若 Chrome 阻止，请确认扩展已启用。
- 导出文件通过 `chrome.downloads.download` 触发，请允许浏览器自动下载。
- 在仓库根目录执行 `npm install`，随后运行 `npm run lint` 可用 ESLint 检查脚本质量。
